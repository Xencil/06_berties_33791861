# Berties Books

Sample code for Dynamic Web Applications module.

Uses node.js with Express and EJS.

To install and run:

npm install
node index.js

Routes that need a login before using:
/users/list
/users/logout
/books/list
/books/bookadded

Routes without login:
/users/register
/users/login
/users/registered
/users/loggedin
/books/search
/books/search-result
